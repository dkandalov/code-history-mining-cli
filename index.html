./code-history-mining.sh

What is this?

This is a command line application to analyze and visualize project code history. For example, see visualizations for JUnit, Scala, Ruby on Rails.

Basically, it reads your version control history (git, svn) and produces .csv file. Then reads .csv file and produces standalone .html file with visualizations.

How to use?

Make sure you have Java installed (1.6 or above).
Use this button to download zip file: Download as .zip.
Unzip it and run code-mining.sh (or .bat), e.g. to visualize two years of JUnit history: 

$ git clone https://github.com/junit-team/junit
$ cd junit
$ ../code-mining.sh grab --from 01/01/2012 --to 01/01/2014
$ ../code-mining.sh visualize --csv junit.csv

To see help, run script without any parameters. It should be self-explanatory. If not, please feel free to send a question.

Why all this?

Governments and Facebook spy on us, so it would be nice to spy on someone else too. For example, yourself and your own project. But seriously there is a lot of interesting data in code history and we mostly look at it as a list of commits. This is an attempt to change perspective on what code history could look like.
